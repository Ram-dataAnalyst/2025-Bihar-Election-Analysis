# 2025-Bihar-Election-Analysis
This repository contains a complete project workflow for building an Tableau dashboard that visualizes the Bihar Assembly Election Results 2025 using a ECI dataset. The project replicates real election reporting workflows, including data scraping (Python), data preparation, visualization design, and dashboard storytelling.

**Project Overview**

  **This repository contains:**
	
  A mock election dataset generated in Python
	
  A full Tableau dashboard blueprint (ECI-style)
	
  Custom color palette, calculated fields, and layout files
	
  A complete workflow from Python → CSV → Tableau → Dashboard
	
  The dashboard replicates real-world election analytics, including party performance, vote share, constituency outcomes, winning margins, and candidate rankings.
